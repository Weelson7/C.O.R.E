# C.O.R.E. Branding Quick Reference

Minimal guide for fast implementation and review.

## Core Idea

- Cluster-first identity
- Resilient, observable, zero-trust mindset
- Tone: precise, calm, technical

Tagline:

> The network is the computer. The cluster is the organism.

## Visual Direction

- Deep purple base
- Neon LED accents
- Controlled shimmer, never noisy
- Status colors reserved for system health

## Primary Colors

- Base: `#080812`
- Deep purple: `#1a0a3d`
- Neon accent: `#b44fff`
- Dim accent: `#7a2fb5`
- Text light: `#f1e6ff`

## Status Colors

- Healthy: `#39ff8a`
- Info: `#56c7ff`
- Degraded: `#ffc14d`
- Critical: `#ff4466`

## Motion Rules

- Keep transitions short (about 140ms to 240ms)
- Use motion to explain state changes
- Avoid heavy infinite animation
- Respect reduced motion preferences

## UI Do / Do Not

Do:

- Keep cards and panels dark purple with subtle glow
- Make degraded and down states obvious
- Keep labels operational and clear

Do not:

- Use red as decorative color
- Add constant flashing effects
- Hide incidents behind soft language

## Copy Style

Use:

- Online
- Degraded
- Failover active
- Manual action required

Avoid:

- Oops
- Uh oh
- Something broke
